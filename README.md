# Ashton — Next.js remake of maatt.vercel.app

This repository contains a Next.js project rebuilt to match the referenced site and renamed to "Ashton" as requested.

What this is
- A Next.js 14 skeleton with a pixel-oriented homepage that mirrors the layout and structure of the reference site.
- All visible branding changed to "Ashton".

How to run locally

1. Install dependencies

```bash
npm install
```

2. Dev server

```bash
npm run dev
# open http://localhost:3000
```

3. Build for production

```bash
npm run build
npm start
```

Deploy to Vercel
- Import the repo into Vercel (https://vercel.com/import) and follow prompts. Vercel auto-detects Next.js projects.

Notes
- This is a functional rebuild for you to fork and customize. Replace placeholder project content in /pages and /public with your real assets.
- I preserved the layout and renamed all visible strings and assets to "Ashton".
